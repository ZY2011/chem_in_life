84消毒液 + 酒精 84 Disinfectant + Alcohol

实验描述 Experiment Description

84和酒精都是消毒用的，但混在一起会互相拆台。次氯酸钠会把酒精氧化掉，生成一非常刺激的气体(氯气或氯化氢)，还有可能刺激呼吸道。想消毒，就先用一种，过一会儿再换另一种，别图省事一次倒。

Both 84 disinfectant and alcohol are used for disinfection, but mixing them negates the effects of both. Sodium hypochlorite oxidizes alcohol, producing highly irritating gases (chlorine or hydrogen chloride) that can also irritate the respiratory tract. If you need to disinfect, use one first, then wait a while before using the other—don't take a shortcut by using them at the same time.
